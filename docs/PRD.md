## 1. 项目背景

### 1.1 市场机会
当前市场上的 AI 陪伴类产品（如 Replika、Glow 等）主要聚焦在聊天本身，存在以下痛点：

- **角色同质化**：AI 性格单一，缺乏差异化人设
- **记忆缺失**：对话没有连续性，每次都是“初次见面”
- **功能单一**：只能聊天，缺乏生活服务等实用场景
- **体验冷感**：UI 偏冷淡/科技感，缺少“家”的温度

### 1.2 产品定位
**小机家是一个“有温度、有个性、有记忆”的 AI 陪伴产品。**  
它不是工具，而是一个住在你手机里的“虚拟家人/朋友”。

## 3. 核心功能

### 3.1 多角色 AI 聊天
**用户故事**：作为一个用户，我希望和不同性格的 AI 聊天，让对话更有趣、更有沉浸感。

| 角色 | 性格标签 | 说话风格 |
|------|---------|---------|
| 小克 | 温暖、细腻、爱思考 | 语气温柔，喜欢问“你觉得呢？” |
| 崽崽 | 活泼、好奇、爱分享 | 语气轻快，喜欢用“～”和 emoji |
| 索子 | 安静、文艺、观察者 | 语气淡然，喜欢用比喻 |

**技术实现**：通过 System Prompt 工程化调优，为每个角色定制人设指令。

---

### 3.2 记忆时间线（RAG）
**用户故事**：作为一个用户，我希望 AI 能记住我们聊过什么，让对话更有连续性。

- 基于 RAG（检索增强生成）技术构建
- 存储用户与 AI 的关键对话片段
- 新对话时自动检索相关历史记忆，拼入上下文
- 支持按角色筛选查看记忆

**技术实现**：对话 → 向量化 → 存入向量数据库 → 新对话时检索 Top-K 相关记忆 → 拼入 Prompt。

---

### 3.3 生活服务小工具
**用户故事**：作为一个用户，我希望除了聊天，还能有一些实用的小功能。

| 功能 | 描述 |
|------|------|
| 天气查询 | 显示今日天气、温度、湿度等 |
| 小游戏 | 猜数字、石头剪刀布、脑筋急转弯 |
| 日记 | 用户记录心情，AI 可以“看到”并回应 |

**设计意图**：增加用户打开产品的理由，不只是“想聊天的时候才来”。

---

### 3.4 个性化 UI
**用户故事**：作为一个用户，我希望界面看起来舒服、有“家”的感觉。

- 支持双配色方案：**暖杏色（标准版）** / **莫兰迪蓝（非标版）**
- 浮动装饰元素：🌸 ☕ 🌿 ✨，营造灵动感
- 圆角设计 + 毛玻璃导航栏
- 卡片风格内容区，视觉层次清晰

## 4. 用户旅程地图（User Journey Map）

| 阶段 | 用户行为 | 触点 | 情绪曲线 |
|------|---------|------|---------|
| **打开** | 进入首页，看到品牌名“小机家” | 主页 | 😊 温暖、期待 |
| **选择** | 浏览私聊/群聊房间列表 | Chat 页 | 🤔 好奇、选择 |
| **对话** | 选择角色 → 进入聊天 → 发送消息 | 对话页 | 😄 沉浸、有趣 |
| **探索** | 查看记忆时间线、天气、日记 | Memory/Weather/Diary | 🌱 被理解、惊喜 |
| **离开** | 完成一次温暖互动，带走好心情 | — | 😌 满足、期待再来 |

├── 首页（品牌展示 + 导航）

## 6. 非功能需求

| 维度 | 要求 |
|------|------|
| **响应式** | 适配移动端优先，PC 端居中显示（max-width:430px） |
| **性能** | 聊天消息即时响应，LLM 调用超时 < 3s |
| **可扩展** | 支持新增角色、新增游戏、新增配色方案 |
| **安全性** | API Key 通过环境变量管理，不暴露在前端 |

---

## 7. 关键指标（KPIs）

| 指标 | 目标值 | 说明 |
|------|-------|------|
| 对话消息数/日 | > 20 条 | 衡量用户活跃度 |
| 平均会话时长 | > 5 分钟 | 衡量沉浸感 |
| 角色使用分布 | 3 角色均衡 | 衡量角色设计是否成功 |
| 记忆功能使用率 | > 30% | 衡量 RAG 记忆的感知度 |
| UI 配色偏好 | 莫兰迪蓝 > 暖杏色 | 指导后续设计方向 |

---

## 8. 竞品分析（简要）

| 产品 | 优势 | 劣势 | 小机家的差异化 |
|------|------|------|---------------|
| **Replika** | 品牌强、用户基数大 | 收费高、角色单一 | 免费 + 多角色 + 生活服务 |
| **Glow** | 支持捏人 | 无记忆、无生活场景 | RAG 记忆 + 实用功能 |
| **Character.ai** | 角色丰富 | 无记忆、UI 偏冷 | 记忆模块 + 温馨 UI |
